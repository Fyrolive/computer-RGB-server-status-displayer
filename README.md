# Tutorial to display anything you want on your ram

## Purpose

My ram had no built in color cycles, so i created them myself (+displayed my servers status in it).


## Requirements & Disclaimers

In this tutorial, i'll show how to display anything on your ram using [openRGB](https://openrgb.org).

I'll be using the build in openRGB CLI. You will need to fully install openRGB before beeing able to follow this tuto.

This step by step guide will be covered on **LINUX** only. Good luck for windows users lol.

Do note that playing with your ram display can completly mess up it's display, so do be carefull when your injecting instructions.
